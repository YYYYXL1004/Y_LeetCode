# 格式参考文件

本文件包含各文件的实际格式模板，供 Skill 执行时参考。

## 1. daily/YYYY-MM.md 格式

### 文件头
```markdown
# YYYY年M月 学习记录
```

### 每日记录条目（日期降序，最新在最前）
```markdown
## M.DD
- 专题方向概要（如"动态规划入门专题"）
  - 题号：题目名，核心算法描述
    - 关键思路/状态定义/转移方程
    - 坑点：xxx
  - 题号：题目名，核心算法描述
    - 关键思路
    - 坑点：xxx
- 详细笔记见 [专题名](../notes/专题名.md)

|| 题号 | 题目 | 难度 | 状态 |
||------|------|------|------|
|| 70 | 爬楼梯 | 简单 | ✅ |
|| 322 | 零钱兑换 | 中等 | ✅ |

---
```

### 文件末尾模板注释
```markdown
<!-- 模板
## M.XX
- 今日学习内容概要
- 详细笔记见 [专题名](../notes/专题名.md)

|| 题号 | 题目 | 难度 | 状态 |
||------|------|------|------|
|| | | | |
-->
```

## 2. README.md 格式

### 学习进度表
```markdown
| 月份 | 刷题数量 | 详情 |
|------|----------|------|
| 2025.01 | 47 | [查看](./daily/2025-01.md) |
| 2025.02 | 122 | [查看](./daily/2025-02.md) |
| 2025.03 | 45 | [查看](./daily/2025-03.md) |
```

### 题目分类表（力扣 + 竞赛混排，按题号排序）
```markdown
### 动态规划
| 题号 | 题目 | 难度 | 备注 |
|------|------|------|------|
| 70 | 爬楼梯 | 简单 | DP入门，dp[i]=dp[i-1]+dp[i-2] |
| 72 | 编辑距离 | 困难 | 二维DP，插入/删除/替换三选一 |
```

### 天梯赛 L1 题目表

|------|------|------|

## 3. notes/专题名.md 格式

### 题目解析
```markdown
## 题目名（题号）
**核心思路/状态定义**：一句话概括

**状态转移**：`dp[i] = ...`（DP题特有）

### 方法一：迭代解法（自底向上）
代码块

### 方法二：备忘录递归（自顶向下）
代码块（如有）

---
```

### 相关题目表格（笔记末尾）
```markdown
## 相关题目
| 题号 | 题目 | 难度 |
|------|------|------|
| 70 | 爬楼梯 | 简单 |
```

## 4. 难度等级对照

| 来源 | 难度标记 |
|------|---------|
| 力扣 | 简单 / 中等 / 困难 |
| 天梯赛 | L1 / L2 / L3 |
| 蓝桥杯 | 对应等级 |
| 校赛 | 沿用对应分组等级 |

## 5. 文件命名规范

| 来源 | 文件命名 | 示例 |
|------|---------|------|
| 力扣 | `题号.题目名.cpp` | `70.爬楼梯.cpp` |
| 天梯赛 | `Lx-xxx.cpp` 或 `XB-Lx-xxx.cpp` | `L2-001.cpp`、`XB-L2-1.cpp` |
| 蓝桥杯 | `题号.题目名.cpp` | `3513.岛屿个数.cpp` |
