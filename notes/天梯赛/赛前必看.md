# 天梯赛考前速查：常见技巧与坑点

> 考前 30 分钟快速过一遍，查漏补缺

---

## 一、输入输出（L1 送命重灾区）

1. `cin >> n` 后接 `getline` 必须先 `getchar()` 吃换行，否则 `getline` 读到空串
2. 混合输入：`cin >> A; getchar(); getline(cin, B);` 处理"第一个空格后全是B"的情况
3. ID 输出用 `printf("%05d", id)` 补前导零，天梯赛链表题、学号题几乎必考这个格式
4. 行末空格：`if(i > 0) cout << " ";` 先判断再输出，或者先输出第一个元素再循环输出 `" " + rest`
5. 小数精度：`printf("%.2f", x)` 或 `cout << fixed << setprecision(2)`

---

## 二、数据类型（最隐蔽的 WA）

1. 看到"长整型范围"或数据 > 2×10⁹ 立刻用 `long long`
2. 中间结果溢出：`long long c = (long long)a * b;` 先转再乘，不是先乘再赋值
3. `stoi` 只能转 int 范围，超出用 `stoll`
4. `size_t` 是无符号的，`s.find("x") < 0` 永远为 false，必须用 `!= string::npos` 或 `!= s.npos`（两者等价）。`find` 参数可以是字符串也可以是字符：`s.find("x")` 和 `s.find('x')` 都行
5. `vector.size()` 返回 `size_t`，`v.size() - 1` 在空 vector 上会下溢变成巨大正数。避免方法：竞赛里最省心直接 `(int)v.size()` 强转成有符号数；或者先 `if(!v.empty())` 再减；或者循环条件写 `i + 1 < v.size()` 代替 `i < v.size() - 1`

## 三、STL 高频坑点

| 坑 | 错误写法 | 正确写法 |
|---|---|---|
| stack/queue 的 pop | `x = st.pop()` | `x = st.top(); st.pop();` |
| map 访问自动创建 | `if(m["x"] == 0)` | `if(m.count("x"))` |
| set 不能下标 | `s[0]` | `*s.begin()` |
| 遍历时删除 | `for(it) { erase(it); }` 迭代器失效 | `it = v.erase(it);` 删了不 `it++`，没删才 `it++` |
| reverse/sort 区间 | 闭区间 `[l,r]` 直接用 | 要 `r+1`，左闭右开 |
| priority_queue 小根堆 | 忘写模板参数 | `priority_queue<int, vector<int>, greater<int>>` |
| priority_queue 自定义比较 | lambda 直接塞模板 | 见下方写法 |

**priority_queue 自定义比较写法：**
```cpp
auto cmp = [](const pair<int,int>& a, const pair<int,int>& b) {
    return a.second > b.second; // 大的在堆底， 按 second 小的优先出来
};
priority_queue<pair<int,int>, vector<pair<int,int>>, decltype(cmp)> pq(cmp);
```

## 四、L1 拿满分技巧（100分保底）

- L1 全是模拟/字符串/简单数学，难度不高但格式要求极严
- 字符串替换四板斧：

**1. find + replace（简单替换）**
```cpp
size_t pos = 0;
while ((pos = s.find("old", pos)) != string::npos) {
    s.replace(pos, 3, "new");  // 3 是 "old" 的长度
    pos += 3;                  // 跳过替换后的部分，防死循环
}
```

**2. 线性扫描重建（防连环污染）**
```cpp
string res;
for (size_t i = 0; i < s.length(); ) {
    if (s.substr(i, 3) == "old") {
        res += "new";
        i += 3;
    } else {
        res += s[i];
        i++;
    }
}
```

**3. regex_replace（独立单词替换）**
```cpp
#include <regex>
string res = regex_replace(s, regex("\\bold\\b"), "new");
// \b 是单词边界，只替换独立单词，不会误伤 "bold" 里的 "old"
```

**4. stringstream 按单词处理**
```cpp
stringstream ss(s);
string word, res;
while (ss >> word) {
    if (word == "old") word = "new";
    if (!res.empty()) res += " ";
    res += word;
}
// 适合按空格分词后逐词判断/替换的场景
// 注意：会丢失多余空格，只保留单空格分隔
```
- 替换顺序很重要：先替换长串再替换短串，避免新产生的子串被二次替换
- 独立单词判断：首尾加空格 `s = ' ' + s + ' '`，然后检查前后是否为分隔符
- 字符判断函数记牢：`isdigit`, `isalpha`, `isalnum`, `isspace`, `toupper`, `tolower`

## 五、L2 核心算法模板（必须烂熟）

1. **并查集**（几乎每年必考）：路径压缩 + 按秩合并，带权并查集处理敌友关系
2. **二叉树遍历重建**：后序+中序 → 建树 → 层序输出，这是天梯赛经典套路
3. **Dijkstra**：单源最短路 + 路径计数 + 前驱数组重建路径 + 多约束松弛
4. **Floyd**：N ≤ 500 时用邻接矩阵，三重循环
5. **静态链表**：用 `data[]` + `next[]` 数组模拟，地址是 5 位整数，输出 `%05d`
6. **栈模拟**：混洗判断、表达式求值（双栈）、队列+栈综合

### 以下是你还没刷到但天梯赛考过/可能考的专题：

7. **简单数学**（L1/L2 常考）
   - GCD/LCM：`__gcd(a, b)`，LCM 先除后乘防溢出 `a / __gcd(a, b) * b`
   - 素数判断：`i * i <= n` 循环试除
   - 快速幂：`qpow(a, b)` 求 `a^b % MOD`
   - 进制转换：十进制转 k 进制（取余倒序）、k 进制转十进制（逐位累乘）
   - 分数运算：通分 → 运算 → 约分，分母为负要调整符号
   - 坑点：`pow()` 返回 double 有精度问题，整数幂运算用快速幂或手写循环；LCM 先除再乘不然中间溢出

8. **BFS 求连通分量 / 最短路（无权图）**
    - 天梯赛图论题很多本质是 BFS：L2-048 寻宝图（网格连通块）、L3-004 肿瘤诊断（3D BFS）、L3-008 喊山（无权最远距离）
    - 网格 BFS 模板：四方向 `dx[] = {-1,1,0,0}`，标记 `vis[][]`，队列存坐标

9. **大模拟题生存指南**
    - L2 每年至少 1 道大模拟（L2-034 口罩发放、L2-030 冰岛人等），代码量大、边界多
    - 策略：先通读题目标记所有输出格式要求 → 用样例手动模拟一遍 → 先写框架再填细节
    - 常见 WA 原因：题意理解偏差、多条件优先级判断错误、浮点精度（能用整数就用整数）、排序比较函数不满足严格弱序

## 六、L2 常见陷阱

- 链表题：地址用 `%05d` 输出，末尾 `-1` 不补零；可能有孤立节点不在链表上
- 并查集题：`find` 之前别忘初始化 `fa[i] = i`；统计时要对每个节点都 `find` 一次
- 树题：完全二叉树用数组下标 `2*u` 和 `2*u+1` 直接映射，省去建树
- 图着色题：检查每条边两端颜色不同，不是检查每个节点
- 大模拟题：仔细读题，边界 case 是主要失分点，先写框架再填细节

---

## 七、L3 得分策略

- L3 图论占 35.7%，最短路变种是重中之重
- 多约束 Dijkstra：松弛时按优先级依次比较（距离 → 点权 → 路径数等）
- 能拿部分分就拿：L3 很多题可以暴力拿一部分分，不要空着
- 树状数组求在线中位数（L3-002 特殊堆栈）是经典模板

### 并查集模板
```cpp
int parent[MAXN];

void init(int n) {
    for(int i = 0; i <= n; i++) parent[i] = i;
}

int find(int x) {
    if(parent[x] != x) parent[x] = find(parent[x]);
    return parent[x];
}

void _union(int p, int q) {
    int rp = find(p), rq = find(q);
    if(rp != rq) parent[rp] = rq;
}
```

### Dijkstra 模板（带路径计数）
```cpp
const int inf = 0x3f3f3f3f;

int n, m, s, ed;
vector<int> dist, paths, total, vpre, team;
vector<vector<pair<int,int>>> adj;

void dijkstra() {
    dist[s] = 0; paths[s] = 1;
    total[s] = team[s];
    priority_queue<pair<int,int>, vector<pair<int,int>>, greater<pair<int,int>>> pq;
    pq.push({0, s});
    while(!pq.empty()) {
        auto cur = pq.top(); pq.pop();
        int d = cur.first, u = cur.second;
        if(d > dist[u]) continue;
        for(auto& edge : adj[u]) {
            int v = edge.first, w = edge.second;
            if(dist[u] + w < dist[v]) {
                dist[v] = dist[u] + w;
                paths[v] = paths[u];
                total[v] = total[u] + team[v];
                vpre[v] = u;
                pq.push({dist[v], v});
            } else if(dist[u] + w == dist[v]) {
                paths[v] += paths[u];
                if(total[u] + team[v] > total[v]) {
                    total[v] = total[u] + team[v];
                    vpre[v] = u;
                }
            }
        }
    }
    // 路径还原
    stack<int> path;
    for(int i = ed; i != -1; i = vpre[i]) path.push(i);
    cout << path.top(); path.pop();
    while(!path.empty()) {
        cout << " " << path.top();
        path.pop();
    }
}
```

### 二叉树遍历重建（后序+中序 → 层序）
```cpp
vector<int> postorder, inorder;
unordered_map<int, int> inIndex;
int lch[1005], rch[1005];

int build(int postL, int postR, int inL, int inR) {
    if(postL > postR) return 0;
    int root = postorder[postR];
    int k = inIndex[root];
    int leftSize = k - inL;
    lch[root] = build(postL, postL + leftSize - 1, inL, k - 1);
    rch[root] = build(postL + leftSize, postR - 1, k + 1, inR);
    return root;
}

// BFS 层序遍历
void levelOrder(int root) {
    queue<int> q;
    q.push(root);
    bool first = true;
    while(!q.empty()) {
        int u = q.front(); q.pop();
        if(!first) cout << " ";
        cout << u;
        first = false;
        if(lch[u]) q.push(lch[u]);
        if(rch[u]) q.push(rch[u]);
    }
}
```

const int MOD = 1e9+7;
long long qpow(long long a, long long b) {
    long long res = 1;
    a %= MOD;
    while(b > 0) {
        if(b & 1) res = (res * a) % MOD;
        a = (a * a) % MOD;
        b >>= 1;
    }
    return res;
}

**进制转换模板（推荐背）**
```cpp
// 支持 2~36 进制
const string DIGITS = "0123456789ABCDEFGHIJKLMNOPQRSTUVWXYZ";

int charToVal(char c) {
    if(c >= '0' && c <= '9') return c - '0';
    if(c >= 'A' && c <= 'Z') return c - 'A' + 10;
    if(c >= 'a' && c <= 'z') return c - 'a' + 10;
    return -1;
}

// 十进制 -> k 进制（取余倒序）
string decToBase(long long n, int k) {
    if(k < 2 || k > 36) return "";
    if(n == 0) return "0";

    bool neg = (n < 0);
    if(neg) n = -n;

    string s;
    while(n > 0) {
        int r = n % k;
        s += DIGITS[r];   // 先得到低位
        n /= k;
    }
    reverse(s.begin(), s.end()); // 逆序后才是正常表示
    if(neg) s = "-" + s;
    return s;
}

// k 进制 -> 十进制（逐位累乘）
// 例："1A"(16) => ((0*16+1)*16+10)
long long baseToDec(const string& s, int k) {
    if(k < 2 || k > 36) return -1;
    int i = 0;
    bool neg = false;
    if(!s.empty() && s[0] == '-') {
        neg = true;
        i = 1;
    }

    long long x = 0;
    for(; i < s.size(); i++) {
        int v = charToVal(s[i]);
        if(v < 0 || v >= k) return -1; // 非法字符
        x = x * k + v;
    }
    return neg ? -x : x;
}
```

**分数运算**
```cpp
void simplify(long long& num, long long& den) {
    if(den < 0) {
        num = -num;
        den = -den;
    }
    long long g = __gcd(llabs(num), den);
    num /= g;
    den /= g;
}

// a/b + c/d = num/den（输出已约分）
void addFrac(long long a, long long b, long long c, long long d,
             long long& num, long long& den) {
    den = b / __gcd(b, d) * d;          // 先除后乘防溢出
    num = a * (den / b) + c * (den / d);
    simplify(num, den);
}
```

### BFS 网格连通块模板（寻宝图/肿瘤诊断类）
```cpp
int grid[MAXN][MAXN], vis[MAXN][MAXN];
int dx[] = {-1, 1, 0, 0};
int dy[] = {0, 0, -1, 1};
int n, m;

int bfs(int sx, int sy) {
    queue<pair<int,int>> q;
    q.push({sx, sy});
    vis[sx][sy] = 1;
    int cnt = 0;
    while(!q.empty()) {
        auto [x, y] = q.front(); q.pop();
        cnt++;
        for(int d = 0; d < 4; d++) {
            int nx = x + dx[d], ny = y + dy[d];
            if(nx >= 0 && nx < n && ny >= 0 && ny < m
               && !vis[nx][ny] && grid[nx][ny] == 1) {
                vis[nx][ny] = 1;
                q.push({nx, ny});
            }
        }
    }
    return cnt; // 连通块大小
}
```
