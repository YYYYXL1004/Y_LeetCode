## 一、输入输出陷阱

### 1.1 字符串读取

**问题**：`cin >> s` 遇到空格就停止

```cpp
// 输入：Hello World
string s;
cin >> s;  // s = "Hello"，World 丢失
```

**解决**：使用 `getline`

```cpp
string s;
getline(cin, s);  // s = "Hello World"
```

**注意**：`cin >> n` 后接 `getline` 要先吃掉换行符

```cpp
int n;
cin >> n;
getchar();  // 吃掉换行符
getline(cin, s);
```

### 1.2 混合输入

**问题**：空格分隔的输入，第一个空格后都是 B

```cpp
// 输入：123 abc def
// A = "123", B = "abc def"（B 可能包含空格！）
```

**解决**：

```cpp
string A, B;
cin >> A;           // 读取第一个非空格串
getchar();          // 吃掉 A 后面的空格
getline(cin, B);    // 读取整行剩余部分
```

### 1.3 数字转字符串

```cpp
// 补前导0输出（如 ID 位数固定）
int id = 123;
printf("%05d", id);  // 输出 "00123"

// 字符串转数字
string s = "12345";
int n = stoi(s);      // int
long long m = stoll(s); // long long（注意范围！）
```

## 二、数据类型陷阱

### 2.1 范围溢出

| 类型 | 范围 |
|------|------|
| `int` | -2×10⁹ ~ 2×10⁹ |
| `long long` | -9×10¹⁸ ~ 9×10¹⁸ |

**常见问题**：
- 题目说"长整型范围内" → 用 `long long` 19位
- `stoi` 范围有限，超出用 `stoll`
- 中间计算可能溢出，即使结果在范围内

```cpp
// 危险：中间结果溢出
int a = 1e9, b = 1e9;
long long c = a * b;  // 先溢出再赋值，c 是错误的值

// 安全：提前转 long long
long long c = (long long)a * b;
```

### 2.2 负数处理

```cpp
// __gcd 可以处理负数，结果为正
__gcd(-12, 8);  // 返回 4

// abs 对整数取绝对值
abs(-5);        // 返回 5

// fabs 对浮点数取绝对值
fabs(-3.14);    // 返回 3.14
```

## 三、常用数据结构

### 3.1 字符串操作

```cpp
string s = "Hello World";

// 查找
int pos = s.find('o');      // 返回第一次出现的位置
int pos = s.find("World");  // 返回子串位置，找不到返回 string::npos

// 截取
string sub = s.substr(0, 5);  // 从位置0开始截取5个字符

// 拼接
s += "!";           // 末尾添加
s.push_back('!');   // 末尾添加字符

// 遍历
for(char c : s) { }
for(int i = 0; i < s.size(); i++) { }

// 判断字符类型
isdigit('5');   // 是否数字
isalpha('a');   // 是否字母
isalnum('a');   // 是否数字或字母
isspace(' ');   // 是否空白字符
toupper('a');   // 转大写
tolower('A');   // 转小写
```

### 3.2 集合 set / unordered_set

```cpp
// set：自动排序，O(log n)
set<int> s;
s.insert(3);
s.erase(3);
s.count(3);     // 存在返回1，不存在返回0
s.find(3);      // 返回迭代器，不存在返回 s.end()

// unordered_set：哈希实现，O(1) 平均
unordered_set<int> us;
us.insert(3);
us.count(3);    // 判断是否存在
```

### 3.3 映射 map / unordered_map

```cpp
// map：自动按 key 排序，O(log n)
map<string, int> m;
m["apple"] = 5;
m.insert({"banana", 3});
m.count("apple");  // 判断 key 是否存在

// unordered_map：哈希实现，O(1) 平均
unordered_map<string, int> um;
um["apple"] = 5;
```

### 3.4 向量 vector

```cpp
vector<int> v;
v.push_back(1);     // 末尾添加
v.pop_back();       // 删除末尾
v.size();           // 大小
v.empty();          // 是否为空
v.clear();          // 清空

// 初始化
vector<int> v(10);       // 10个0
vector<int> v(10, 5);    // 10个5
vector<int> v = {1,2,3}; // 初始化列表

// 排序
sort(v.begin(), v.end());
sort(v.begin(), v.end(), greater<int>());  // 降序

// 去重（先排序）
sort(v.begin(), v.end());
v.erase(unique(v.begin(), v.end()), v.end());
```

### 3.5 队列 queue / 优先队列 priority_queue

```cpp
// 队列：FIFO
queue<int> q;
q.push(1);
q.pop();
q.front();   // 队首
q.back();    // 队尾

// 优先队列：默认大顶堆
priority_queue<int> pq;           // 大顶堆
priority_queue<int, vector<int>, greater<int>> pq;  // 小顶堆
pq.push(3);
pq.top();     // 堆顶（最大/最小）
pq.pop();
```

### 3.6 栈 stack

```cpp
stack<int> st;
st.push(1);
st.pop();
st.top();     // 栈顶
st.empty();
```

```cpp
// 绝对值
```

### 4.2 排序与查找

```cpp
// 排序
sort(v.begin(), v.end());
sort(v.begin(), v.end(), greater<int>());  // 降序

// 自定义排序
sort(v.begin(), v.end(), [](int a, int b) {
    return a > b;  // 降序
});

// 二分查找（必须在有序序列上使用）
binary_search(v.begin(), v.end(), 5);  // 是否存在

// 边界查找
lower_bound(v.begin(), v.end(), 5);  // 第一个 >= 5 的位置
upper_bound(v.begin(), v.end(), 5);  // 第一个 > 5 的位置

// 去重
unique(v.begin(), v.end());  // 返回去重后的尾后迭代器
```

### 4.3 全排列

```cpp

do {
    // 处理当前排列
} while(next_permutation(v.begin(), v.end()));
```

## 六、常用模板

### 6.1 GCD / LCM

```cpp
int gcd(int a, int b) {
    return b == 0 ? a : gcd(b, a % b);
}

int lcm(int a, int b) {
    return a / gcd(a, b) * b;  // 先除后乘防溢出
}
```

### 6.2 进制转换

```cpp
// 十进制转其他进制
string toBase(int n, int base) {
    if(n == 0) return "0";
    string s;
    while(n > 0) {
        int r = n % base;
        s = (char)(r < 10 ? '0' + r : 'A' + r - 10) + s;
        n /= base;
    }
    return s;
}

// 其他进制转十进制
int toDecimal(string s, int base) {
    int result = 0;
    for(char c : s) {
        result = result * base + (isdigit(c) ? c - '0' : c - 'A' + 10);
    }
    return result;
}
```

### 6.3 判断素数

```cpp
bool isPrime(int n) {
    if(n < 2) return false;
    for(int i = 2; i * i <= n; i++) {
        if(n % i == 0) return false;
    }
    return true;
}
```

### 6.4 分数运算

```cpp
struct Fraction {
    long long num, den;  // 分子，分母

    void simplify() {
        if(den < 0) { num = -num; den = -den; }
        long long g = __gcd(abs(num), den);
        num /= g; den /= g;
    }

    Fraction operator+(const Fraction& other) {
        Fraction result;
        result.den = den / __gcd(den, other.den) * other.den;  // LCM
        result.num = num * (result.den / den) + other.num * (result.den / other.den);
        result.simplify();
        return result;
    }
};
```

### 7.2 常见错误

| 错误 | 原因 | 解决 |
|------|------|------|
| RE (Runtime Error) | 数组越界、除以零 | 检查下标范围、特判除数 |
| TLE (Time Limit) | 算法复杂度过高 | 优化算法、减少循环 |
| WA (Wrong Answer) | 逻辑错误 | 检查边界、输出格式 |
| MLE (Memory Limit) | 内存超限 | 减少数组大小、使用更小的数据类型 |
