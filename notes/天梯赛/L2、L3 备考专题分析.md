## 专题一：栈和队列（共7题）

**核心考点**：栈模拟（混洗、调度）、双栈表达式求值、队列+栈综合模拟、特殊栈（中值维护）。

**难度递进**：L2偏基础栈/队列模拟与判断，L3偏高级操作维护。

### 刷题路线

```
入门 → L2-032（纯栈混洗）→ L2-033（双栈计算器）→ L2-045（栈模拟）

```

## 专题二：链表模拟（共2题）

**核心考点**：数组模拟静态链表（data[] + next[]）、节点删除、链表重排、反转、头节点处理。

**重要提示**：天梯赛全部用静态链表（地址为00000~99999的5位整数）模拟，不支持真实指针。

### 刷题路线

```
L2-002（去重+输出删除链表）→ L2-022（重排+反转综合）
```

### L2 级别（2题）

| 状态 | 题号 | 题目名称 | 分数 | 核心考点 |
|------|------|----------|------|----------|
| [ ] | L2-002 | 链表去重 | 25 | 删除数据绝对值相同的节点，同时输出剩余链表和被删除节点链表 |
| [ ] | L2-022 | 重排链表 | 25 | 将链表按 L1→Ln→L2→Ln-1… 的方式重排，输出新链表 |

### 常见坑点

- **L2-002**：必须同时输出被删除的节点链；绝对值去重而非数值去重；头节点地址可能变化。
- **L2-022**：先用vector存所有地址 → 奇偶分离 → 后半反转 → 交叉合并；边界处理（节点数≤2、奇数长度）。
- 输出格式严格：每个节点一行 `地址 data next`，地址用 `%05d` 输出，-1表示空。

### 备注

- 堆相关（如L2-012）和并查集题常被归入"树"专题，因为底层就是树结构。
- 每年天梯赛还会出现树上距离、外卖路径等变种。
- 4.16 模板补充：
  1. 父数组建树后，`BFS` 可稳定拿层深/最深层成员，`DFS` 可稳定拿根到叶最优路径（先排序邻接表可直接保证字典序）。
  2. 完全二叉树题优先考虑“数组下标即结构”：`u -> 2u / 2u+1`，可把“重建 + 层序输出”压到同一份数组里做。
  3. 在线中位数类“栈 + 查询”可拆成“顺序结构（栈）+ 频次结构（树状数组）”，把 `PeekMedian` 降到对数级。

**核心考点**：路径压缩+按秩合并、带权并查集（关系/敌友）、删除模拟（时间戳或倒序处理）、多组统计。

### 常见模板技巧

- **普通DSU**：find路径压缩 + union按秩合并
- **带权**：额外relation[]数组记录与根的关系
- **动态删除**：倒序处理或时间戳标记（L2-013经典坑）
- **统计**：用size[]/sum[]维护每个根的节点数/房产等
