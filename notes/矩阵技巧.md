# 矩阵技巧

## 螺旋矩阵遍历
- 维护四个边界 `upper/lower/left/right`，按顺时针方向遍历并收缩边界

## 矩阵旋转
- 顺时针 90°：沿主对角线镜像翻转 + 反转每一行
- 逆时针 90°（推荐）：沿副对角线翻转 + 反转每一行
- 反转每一行比反转列方便：`for(auto &row : matrix) reverse(row.begin(), row.end());`

## 转置矩阵
- `res[j][i] = matrix[i][j]`，注意维度变化 `m×n → n×m`

## 坐标映射
- 一维 → 二维：`(idx / n, idx % n)`
- 二维 → 一维：`i * n + j`
- 重塑矩阵：一维索引 `i` → 原矩阵 `mat[i/n][i%n]`，新矩阵 `res[i/c][i%c]`

## 对角线识别
- 主对角线：`i - j = 常数`
- 副对角线：`i + j = 常数`
- 按对角线排序：哈希表 + vector + 索引

## 二维网格迁移
- 展平为一维，右移 k 位后映射回二维
  ```cpp
  int newPos = (i * n + j + k) % total;
  res[newPos / n][newPos % n] = grid[i][j];
  ```

## 通用技巧
- 降维处理：二维 ⟺ 一维（适用于规律性操作）
- 遍历顺序：按行、按列、按对角线根据问题选择
- 空间优化：能否原地修改或使用 O(1) 额外空间
