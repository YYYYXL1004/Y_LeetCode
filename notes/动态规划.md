# 动态规划

## 规划兼职工作
- 按结束时间排序 + DP + 二分查找优化
- `dp[i]` 表示考虑前 i 个工作的最大收益
- 转移时二分查找不冲突的上一个工作
- 使用 `upper_bound` 找第一个结束时间 > 当前开始时间的位置，其前一个即为合法状态
  ```cpp
  // 按结束时间排序，提取 ends 数组用于二分
  for(int i = 1; i <= n; i++) {
      int currStart = jobs[i-1][0], currProfit = jobs[i-1][2];
      // 在 ends 的前 i-1 个元素中找第一个 > currStart 的位置
      auto it = upper_bound(ends.begin(), ends.begin() + i - 1, currStart);
      int prevIdx = distance(ends.begin(), it);
      // dp 是 1-based，ends 是 0-based，distance 刚好对应 dp 中上一个合法状态
      dp[i] = max(dp[i-1], dp[prevIdx] + currProfit);
  }
  ```
