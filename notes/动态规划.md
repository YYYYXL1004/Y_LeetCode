# 动态规划

## DP 核心要素
1. **状态定义**：确定 dp 数组/函数的含义，找出变化的变量
2. **选择**：导致状态变化的行为
3. **状态转移方程**：dp[i] 与 dp[i-1] 等之间的关系
4. **base case**：初始条件/边界情况

## 两种实现方式

### 1. 备忘录递归（自顶向下）
- 从目标状态开始，递归求解子问题
- 用备忘录避免重复计算

### 2. DP table 迭代（自底向上）
- 从小问题开始，逐步推导出大问题的解
- 通常更高效，无递归栈开销

## 斐波那契数（509）
**状态转移**：`dp[i] = dp[i-1] + dp[i-2]`

### 方法一：备忘录递归（自顶向下）
```cpp
class Solution {
public:
    vector<int> memo;
    
    int fib(int n) {
        memo = vector<int>(n + 1, -1);  // 初始化为特殊值表示未计算
        return dp(n);
    }
    
    int dp(int n) {
        // base case
        if(n == 0 || n == 1) return n;
        // 查备忘录，避免重复计算
        if(memo[n] != -1) return memo[n];
        // 存入备忘录
        memo[n] = dp(n - 1) + dp(n - 2);
        return memo[n];
    }
};
```

### 方法二：迭代解法（自底向上）
```cpp
int fib(int n) {
    if(n == 0 || n == 1) return n;
    vector<int> dp(n + 1);
    dp[0] = 0, dp[1] = 1;
    for(int i = 2; i <= n; i++) {
        dp[i] = dp[i - 1] + dp[i - 2];
    }
    return dp[n];
}
```

## 零钱兑换（322）
**完全背包问题**：硬币数量无限，求凑出金额的最少硬币数

**状态定义**：`dp[i]` 表示凑出金额 i 所需的最少硬币数

**状态转移**：`dp[i] = min(dp[i], dp[i - coin] + 1)`，对所有 coin 遍历

### 方法一：备忘录递归（自顶向下）
```cpp
class Solution {
public:
    vector<int> memo;
    
    int coinChange(vector<int>& coins, int amount) {
        memo = vector<int>(amount + 1, -666);  // 特殊值表示未计算
        return dp(coins, amount);
    }
    
    // 定义：凑出金额 amount，至少需要 dp(coins, amount) 个硬币
    int dp(vector<int>& coins, int amount) {
        if(amount == 0) return 0;
        if(amount < 0) return -1;
        // 查备忘录
        if(memo[amount] != -666) return memo[amount];
        
        int res = INT_MAX;
        for(int coin : coins) {
            int sub = dp(coins, amount - coin);  // 子问题
            if(sub == -1) continue;  // 子问题无解，跳过
            res = min(res, sub + 1);
        }
        // 存入备忘录
        memo[amount] = (res == INT_MAX) ? -1 : res;
        return memo[amount];
    }
};
```

### 方法二：迭代解法（自底向上）
```cpp
int coinChange(vector<int>& coins, int amount) {
    // 初始化为 amount + 1（不可能的最大值）
    vector<int> dp(amount + 1, amount + 1);
    dp[0] = 0;  // 凑出金额0需要0个硬币
    
    for(int i = 1; i <= amount; i++) {
        for(int coin : coins) {
            if(i - coin < 0) continue;  // 子问题无解
            dp[i] = min(dp[i], dp[i - coin] + 1);
        }
    }
    return (dp[amount] == amount + 1) ? -1 : dp[amount];
}
```

## 规划兼职工作（1235）
- 按结束时间排序 + DP + 二分查找优化
- `dp[i]` 表示考虑前 i 个工作的最大收益
- 转移时二分查找不冲突的上一个工作
- 使用 `upper_bound` 找第一个结束时间 > 当前开始时间的位置，其前一个即为合法状态
  ```cpp
  // 按结束时间排序，提取 ends 数组用于二分
  for(int i = 1; i <= n; i++) {
      int currStart = jobs[i-1][0], currProfit = jobs[i-1][2];
      // 在 ends 的前 i-1 个元素中找第一个 > currStart 的位置
      auto it = upper_bound(ends.begin(), ends.begin() + i - 1, currStart);
      int prevIdx = distance(ends.begin(), it);
      // dp 是 1-based，ends 是 0-based，distance 刚好对应 dp 中上一个合法状态
      dp[i] = max(dp[i-1], dp[prevIdx] + currProfit);
  }
  ```
